Gets Highfly to notify of not only public but also private published content.

== Description ==

Extends the [Highfly plugin](https://wordpress.org/plugins/highfly/). Adds notifications for privately published posts and for comments to private posts and pages.

== Installation ==

1. Install Highfly.
1. Install this.
